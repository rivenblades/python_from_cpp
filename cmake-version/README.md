# Step1
## Setup libraries and build folder
```bash
python setup.py
```

## Alternatively do it yourself

# Step1-alternative
## Clone pybind11
```bash
git clone https://github.com/pybind/pybind11.git
```

## Make build folder
```bash
mkdir build
cd build
```


# Step2
## cd to build foldler
```bash
cd build
```

## Build
```bash
cmake .. && make
```

## Run the executable
```bash
cd ../
./main
```
